# Dashboard 7LM - Publicacao

Esta pasta contem a versao estatica do dashboard pronta para hospedagem.

## Estrutura

- `index.html`: pagina principal.
- `styles/dashboard.css`: estilos originais e ajustes responsivos.
- `scripts/app.js`: logica do BI, filtros, graficos e dados embarcados.
- `assets/`: reservado para imagens e arquivos visuais.
- `data/`: reservado para futura extracao dos dados para CSV/JSON.

## Publicar

Use esta pasta (`dashboard_7LM_publicado`) como raiz do projeto em Netlify, Vercel, GitHub Pages ou servidor proprio.

Nao ha etapa de build. Basta hospedar os arquivos estaticos.
